# Skillovix - Mission 1 Implementation Plan

## Overview
Building a production-ready React application with Vite + Tailwind CSS + Zustand for state management.

## Build Order
1. Install dependencies (npm packages + tailwind)
2. Configure tailwind.config.js with custom colors
3. Create global CSS (index.css)
4. Create Zustand store
5. Create data files (roadmaps, projects, careers)
6. Build UI components (Button, Card, Badge, Input, Toast, ProgressBar)
7. Build layout components (Sidebar, MobileNav, Navbar)
8. Build pages (Landing, Login, Register, Assessment)
9. Set up App.jsx routing
10. Clean up main.jsx
